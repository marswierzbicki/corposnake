# General info
Snake type game happening in corporate open space. Navigate using arrows, collect benefits, avoid mails with ASAPs and BUILD YOUR YOUNG AND DYNAMIC TEAM.

# Requirements
1. Python >= 3.5
2. Libraries:
   - pygame
   - json
   - traceback
   - random
   - sys
